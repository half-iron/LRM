#Router setup

1. Mikrotik hat ein konfigurationsprogramm WinBox welches von https://mikrotik.com/download runtergeladen werden kann.
Als alternative man kann die KonfigurationsWeb interface werwenden.

2. MAC ADRESS der WLAN interface sind unten dem schwarzen deckel zu sehen.
 eth: B8:69:F4:08:BC:C3
 wlan: B8:69:F4:08:BC:C5

3. Router reset: reset knopf 5 sec gedrückt halten wärend boot bis LED flashen, dann loslassen.
